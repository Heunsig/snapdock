## Getting Started

1. Run docker 
    ```bash
    docker run -p 80:2022 illasoft/illa-builder:latest
    ```
2. Open the browser and go to [http://localhost](http://localhost) to access ILLA.
3. Use the following default credentials to log in:
    ```
    Email: root
    Password: password
    ```
