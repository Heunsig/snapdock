## Getting Started

1. Start Papra using Docker:
    ```bash
    docker run -d --name papra -p 1221:1221 ghcr.io/papra-hq/papra:latest
    ```
2. Go to [http://localhost:1221](http://localhost:1221) to access the Papra.
